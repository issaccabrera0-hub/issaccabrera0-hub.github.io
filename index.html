🛠️ Troubleshooting Installation Errors

If the installer refuses to run, try these solutions in order:

Run as Administrator

Windows 11 restricts software that writes files to core system folders. Right-click the installer icon and choose Run as administrator. Provide your password if the system prompts you.

Disable Antivirus Temporarily

Some security tools mark the PingPlotter installer as a false positive. Disable your real-time scanning for five minutes during the setup. Always turn it back on as soon as the installation finishes.

Clean Old Files

If you have an older version of the software, uninstall it first. Use the Windows Control Panel to remove old entries. Restart your computer to clear the memory cache before you run the new installer again.
